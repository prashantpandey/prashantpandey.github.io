Real-world graphs are large, skewed, and constantly changing. This project builds a comprehensive system for dynamic graphs — efficient containers, a framework that supports updates and analytics together, and the benchmarks needed to evaluate them.

We introduced Terrace {% cite DBLP:conf/sigmod/PandeyWXB21 %}, a hierarchical container for skewed dynamic graphs, and BYO {% cite DBLP:journals/pvldb/WheatmanDSDLPX24 %}, a unified framework for benchmarking large-scale graph containers. We have also optimized communication for the distributed execution of graph neural networks {% cite DBLP:conf/ipps/KurtYSPS23 %}.
